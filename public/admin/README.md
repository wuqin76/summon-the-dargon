# Dragon 游戏管理后台

这是一个完全独立于游戏的管理后台系统,专为不懂代码的后端运维人员设计。

## 📁 文件说明

- **login.html** - 登录页面
- **index.html** - 管理后台主页面
- **style.css** - 样式文件
- **script.js** - 功能脚本

## ✨ 功能特点

### 1. 数据总览面板

- 显示总用户数、今日新增用户
- 显示总收入和近期支付情况
- 显示游戏场次和抽奖统计
- 显示待处理提现申请

### 2. 用户管理

- 查看所有用户列表
- 查看用户详细信息(余额、抽奖次数、邀请人数等)
- 封禁/解封用户功能
- 支持搜索和分页

### 3. 支付记录

- 查看所有支付记录
- 显示支付状态和使用情况
- 按时间排序查看
- 支持分页浏览

### 4. 游戏记录

- 查看所有游戏会话
- 区分付费和免费模式
- 显示游戏完成状态
- 追踪抽奖获得情况

### 5. 抽奖记录

- 查看所有抽奖记录
- 显示奖金金额和状态
- 标记需要审核的记录
- 审核状态跟踪

### 6. 提现管理

- 查看提现申请列表
- 按状态筛选(待审核、已批准、已完成等)
- 批准/拒绝提现申请
- 查看提现详细信息

## 🎨 界面设计

### 特点

- **简洁易懂**: 使用图标和颜色区分不同功能
- **中文界面**: 所有文字都是中文,易于理解
- **响应式设计**: 支持电脑和平板访问
- **美观现代**: 使用渐变色和卡片设计

### 颜色说明

- 🔵 蓝色 - 用户相关
- 🟢 绿色 - 收入和成功状态
- 🟣 紫色 - 游戏相关
- 🟠 橙色 - 提现和待处理事项

## 🔐 访问方式

1. **登录页面**: `http://your-domain.com/admin/login.html`
2. **管理后台**: `http://your-domain.com/admin/index.html`

## 📝 使用说明

### 登录

1. 访问登录页面
2. 输入管理员 Token (从后端 `/api/auth/login` 接口获取)
3. 勾选"记住我"可以保持登录状态
4. 点击登录按钮

### 数据查看

1. 登录后自动显示数据总览
2. 点击左侧菜单切换不同模块
3. 使用"刷新"按钮更新数据
4. 使用分页按钮浏览更多记录

### 提现处理

1. 进入"提现管理"页面
2. 找到待处理的提现申请
3. 点击"处理"按钮
4. 选择批准或拒绝
5. 填写备注信息
6. 确认操作

### 用户管理

1. 进入"用户管理"页面
2. 查看用户详细信息
3. 点击"封禁"按钮可以封禁违规用户
4. 填写封禁原因后确认

## 🔧 技术说明

### 前端技术

- 纯 HTML + CSS + JavaScript
- 无需额外框架,直接运行
- 响应式布局,适配各种屏幕

### 后端接口

系统使用以下后端 API:

- `/api/admin/dashboard/stats` - 获取统计数据
- `/api/admin/users/list` - 获取用户列表
- `/api/admin/payments/list` - 获取支付记录
- `/api/admin/games/list` - 获取游戏记录
- `/api/admin/spins/list` - 获取抽奖记录
- `/api/admin/payouts/list` - 获取提现列表
- `/api/admin/payouts/approve` - 批准提现
- `/api/admin/users/ban` - 封禁用户

## 💡 常见问题

### Q: 登录失败怎么办?

A: 请检查 Token 是否正确,确保有管理员权限。

### Q: 数据不显示?

A: 点击页面上的"刷新"按钮,或检查网络连接。

### Q: 忘记了如何操作?

A: 每个按钮都有清晰的中文说明,按照提示操作即可。

### Q: 可以在手机上使用吗?

A: 建议在电脑或平板上使用,体验更好。

## 📊 数据说明

### 状态含义

- **待处理/待审核**: 等待管理员处理
- **已批准**: 管理员已批准,等待执行
- **处理中**: 正在处理中
- **已完成**: 操作已完成
- **已拒绝**: 管理员已拒绝

### 数值说明

- **USDT**: 所有金额单位都是 USDT
- **数字格式**: 使用千分位分隔,便于阅读
- **日期格式**: 年-月-日 时:分 格式
