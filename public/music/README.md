# 背景音乐文件夹

请将背景音乐文件命名为 `background.mp3` 并放在此文件夹中。

## 推荐音乐来源：

- YouTube Audio Library（免费无版权）
- Pixabay Music（免费）
- Incompetech（需署名）

## 音乐要求：

- 格式：MP3
- 大小：建议 < 5MB
- 时长：2-3分钟即可（会自动循环）
- 风格：轻松、愉快、印度风格或现代电子音乐

## 示例音乐网站：

- https://pixabay.com/music/
- https://www.youtube.com/audiolibrary
- https://freesound.org/
